# ELAD-SVDSR: Cross-Device Deepfake Detection

**Author:** Kayes Bin Yousuf | Incoming MS ECE, Clarkson University

## Overview
This demo implements a cross-device deepfake detection system
using simulated MFCC features with MC Dropout uncertainty estimation.

## How to Run
Click the **Open in Colab** badge above to run instantly.

## Key Results
- Model trained on AT2020 microphone
- Evaluated across 5 devices (AT2020, SM58, Inni_R1, TOZO_A1, SpyPen)
- MC Dropout abstention on low-confidence predictions
